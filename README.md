# React CV/Portfolio Site

This is a modern React-based portfolio website built with Vite and Tailwind CSS.

## 🚀 Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Install dependencies:

```bash
npm install
```

2. Start the development server:

```bash
npm run dev
```

The application will open at `http://localhost:3000` in your browser.

### Building for Production

```bash
npm run build
```

This creates an optimized build in the `dist/` directory.

### Preview Production Build

```bash
npm run preview
```


## 🎨 Features

- ⚛️ Built with React 18
- ⚡ Vite for fast development and builds
- 🎨 Tailwind CSS for styling
- 📱 Fully responsive design
- 🌙 Dark theme with accent colors
- 🚀 Production-ready

## 🛠️ Tech Stack

- **Frontend**: React, Vite
- **Styling**: Tailwind CSS
- **Build Tool**: Vite
- **Deployment**: Vercel

## 📝 Customization

You can customize:

- Colors in `tailwind.config.js`
- Content in individual page components
- Navigation items in `Header.jsx`
- Font and theme in `index.html`

## 📦 Dependencies

- `react` - UI library
- `react-dom` - React DOM rendering
- `tailwindcss` - Utility-first CSS
- `vite` - Build tool

## 🚀 Deployment

This project is configured for easy deployment on Vercel. Simply connect your GitHub repository to Vercel and it will automatically build and deploy.
